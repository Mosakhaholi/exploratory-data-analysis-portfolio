# Exploratory Data Analysis (EDA) Portfolio

## Overview
This repository contains comprehensive Exploratory Data Analysis (EDA) projects focusing on data cleansing, statistical tracking, and visual storytelling. These projects demonstrate my ability to take raw datasets, clean structural inconsistencies, and deliver data-driven market reports.

---

## Project 1: Automobile Market Performance Report
### Objective
An analytical deep dive into automotive specifications and performance metrics to identify industry trends, correlation factors, and market distributions.

### Key Data Techniques Used:
* **Data Cleansing & Wrangling:** Handled missing data points, verified variable data types, and filtered structural outliers using `Pandas` and `NumPy`.
* **Visual Storytelling:** Developed statistical distribution plots, correlation heatmaps, and scatter plots using `Matplotlib` and `Seaborn` to map feature dependencies (e.g., horsepower vs. fuel efficiency).
* **Deliverables:** Includes both the raw executable Jupyter Notebook and the finalized professional business report summarizing core findings.

## Project 2: Forbes' Richest Athletes Analysis
### Objective
A demographic and financial review of global sports wealth data, tracking income distributions, historical trends, and sports-specific earning power.

### Key Data Techniques Used:
* **Aggregation & Segmentation:** Leveraged grouping (`groupby`) and sorting methods to rank and isolate top-tier earners across historical eras.
* **Trend Analysis:** Tracked the escalation of athletic earnings over time, highlighting the variance between salary and endorsement revenues.
